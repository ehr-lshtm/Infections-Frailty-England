This repository contains analystical code for two manuscripts from the same data set in England.

Manuscript 1: Frailty progression following severe infections in adults ≥65 years in US and England: two matched cohort studies. For this manuscript, analytical code for England component can be found here. Code for the US component can be found in another repository Infections-Frailty-US.

Manuscript 2: A multistate model of frailty progression after severe infections in adults ≥65 years in England: a matched-cohort study.
